二、JS的事件循环

1.什么是js事件循环

在浏览器中出现对应的异步模块，用来提高主线程的执行效率，程序中的异步任务会放在一个队列中，当主线程空闲时从队列中把相应的任务拿出来执行。

2.任务队列

遇到同步任务直接执行，异步任务有两种：

宏队列（macro-task）：

主要有script、setTimeout、setInterval、setImmediate、I/O、UI、rendering

微队列（micro-task）：

主要有promise、process.nextTick、Object.observe(已废弃)、MutationObserver(html5新特性)

3.执行顺序

(1). 首先将 script 代码块入栈，执行 script 代码块

(2). 在执行 script 代码块过程中，如果有异步任务，将其分入对应的宏队列和微队列

(3). 当 script 执行完毕，会先从微队列中把所有的微队列任务拿出来执行

(4). 所有的微队列任务执行完毕后，在执行宏队列的任务
